 GuidePost — WoW Addon Changelog

--------------------------------------------------------------------------------
v2.2.0 - 2026-4-13
--------------------------------------------------------------------------------

NEW FEATURES — UI ENHANCEMENTS
-------------------------------
- Added minimize/maximize button to collapse window to just the title bar
  • Saves screen space while keeping the addon accessible
  • Shows summary line when minimized (tracked count | zone suggestions)
  • Remembers collapsed state between sessions
  • Click minimize button again to restore full window

- Added search bar at the top of the achievement list
  • Filter achievements by name, category, or zone in real-time
  • Case-insensitive search matches as you type
  • Clear button (X) to quickly reset search
  • Filtered results show in all sections (Tracked, In This Zone, All)

NEW COMMAND — /gp next
----------------------
- Added `/gp next` command to set waypoint for next incomplete step
  • Works without opening the UI — perfect for quick navigation
  • Automatically finds the first tracked achievement with incomplete steps
  • Sets TomTom waypoint and displays step description in chat
  • Shows helpful message if no achievements are tracked

UI IMPROVEMENTS
---------------
- Search box tooltip added for better discoverability
- Minimize button tooltip dynamically shows "Minimize" or "Maximize"
- Improved layout handling when minimized/restored

v2.1.0 - 2026-4-13
--------------------------------------------------------------------------------

NEW FEATURES — FACTION SUPPORT
-------------------------------
- Added automatic faction detection system that shows only relevant achievements
  based on your character's faction (Alliance, Horde, or Neutral)
- Implemented faction filtering in achievement lists and zone suggestions
- Added `faction` field to achievement data structure (optional: "Alliance", 
  "Horde", or omit for cross-faction/neutral achievements)

ACHIEVEMENT DATA — HORDE CITIES & ZONES
----------------------------------------
Added comprehensive Horde achievement coverage:
  • Explore Orgrimmar (#1012) — All 6 districts of the capital city
  • Champion of Orgrimmar (#2783) — Argent Tournament representation

  • Let's Do Lunch: Orgrimmar (#5475) — Food tour achievement (5 vendors)
  • Fish or Cut Bait: Orgrimmar (#5477) — Daily fishing quests
  • Explore Mulgore (#736) — Tauren starting zone (12 locations)
  • Explore Thunder Bluff (#1003) — All 4 city platforms
  • Explore Durotar (#43) — Marked as Horde-specific

ACHIEVEMENT DATA — ALLIANCE FACTION TAGS
-----------------------------------------
Updated existing Alliance achievements with faction tags:
  • Stormwind-related achievements (exploration, dailies, fishing)
  • Elwynn Forest exploration and skyriding races
  • Champion/Exalted Champion of Stormwind
  • Alliance-specific zone achievements

CORE FUNCTIONALITY
------------------
- Added GetPlayerFaction() function to detect character faction automatically
- Added IsAvailableForPlayerFaction() to filter achievements by faction
- Updated achievement filtering logic in RefreshZoneSuggestions() and GetAll()
- Neutral achievements (e.g., Pest Control, The Fishing Diplomat) remain 
  visible to all factions

 v1.0.2 — 2026-04-04
--------------------------------------------------------------------------------

BUG FIXES
---------
- Fixed critical crash where GetAchievementCriteriaInfo would throw an error
  for achievements with no Blizzard-registered criteria (e.g. skyriding races),
  silently preventing the main window from opening. All three call sites in
  AchievementData.lua and MainFrame.lua are now wrapped in pcall.
- Set all unverified criteriaIndex values to nil in Achievements.lua. The
  skyriding achievements had criteriaIndex = 1 which caused the crash — these
  must be verified in-game with /gp criteria before being set.

--------------------------------------------------------------------------------
 v1.0.1 — 2026-04-04
--------------------------------------------------------------------------------

BUG FIXES
---------
- Fixed GuidePostNS namespace not being available to other files. Moved creation
  to data/Achievements.lua (first file loaded) instead of GuidePost.lua (last).
- Fixed invalid escape sequences (\C, \U) in texture paths in MinimapButton.lua
  and Tooltip.lua.
- Fixed slash command input parsing. Replaced unreliable input:trim() with
  native Lua pattern match and made all commands case-insensitive.
- Fixed criteria tracking system. Switched from GetAchievementCriteriaInfoByID
  to index-based GetAchievementCriteriaInfo, which works correctly for all
  achievement types. Renamed criteria field to criteriaIndex throughout.
- Fixed duplicate Pest Control entry. [1159] was a leftover with wrong ID and
  bad data — removed entirely.
- Fixed waypoint W button firing multiple times. Added 0.5s click guard.
- Fixed progress bar only filling halfway. Anchored to both TOPLEFT and TOPRIGHT
  and switched to dynamic GetWidth() calculation.
- Updated Interface version from 110002 to 120001 (The War Within Season 2).

NEW FEATURES
------------
- /gp mapid — Prints the correct UiMapID for your current zone to use in
  Achievements.lua.
- /gp criteria <id> — Dumps all criteria with their index numbers for any
  achievement ID.
- /gp scan and /gp scan <zone> — Async zone scanner that finds achievement IDs
  matching a zone name without freezing the game.
- Added .luacheckrc with all WoW API globals defined for VSCode linting.

UI IMPROVEMENTS
---------------
- Step description text now word wraps correctly using TOPRIGHT anchor instead
  of a fixed pixel width.
- Row height is now dynamic based on content length.
- Progress bar now shows "Completed!" at 100% instead of "100%".
- Progress bar stretches full width of the detail panel.

MINIMAP BUTTON
--------------
- Button now orbits the minimap edge on drag instead of moving freely.
- Drag angle is saved to GuidePostDB and restored between sessions.
- Radius calculated dynamically from Minimap:GetWidth() so it works at any
  minimap size.
- Added OnSizeChanged hook so button repositions automatically when the minimap
  is resized in Edit Mode.

TOMTOM INTEGRATION
------------------
- Switched to modern C_Map API (C_Map.GetBestMapForUnit) for mapID resolution.
- Falls back gracefully to the player's current map if no mapID is set, with a
  warning message directing the user to /gp mapid.

ACHIEVEMENT DATA
----------------
- Fixed all achievement IDs. Pest Control was incorrectly assigned ID 1159 —
  corrected to 2556.
- All 18 Pest Control critters added with verified criteriaIndex values from
  in-game testing.
- Removed incorrect "Going to Need a Bigger Bag" Darkmoon fishing entry (wrong
  ID and wrong data).
- Added Memory of Scholomance (18368) with full 11-step ritual guide including
  all candle trade locations and prerequisites.
- Added all 9 Elwynn Forest Flash skyriding achievements (18444-18452) covering
  Normal, Advanced, and Reverse tiers at Bronze, Silver, and Gold.

README
------
- Fixed MD022 markdownlint warnings — all headings now have required blank lines.
- Updated slash command table from /ag to /gp.
- Updated Interface version requirement.
